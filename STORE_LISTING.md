## Detailed description  (max 16,000 chars)

```
Prefer windows over tabs? New Tab, New Window Reloaded automatically moves every new tab into its own separate window — so your browsing lives in windows you can tile, snap, and arrange, instead of an endless strip of tabs.

Flip it on and forget it. Open a link, hit Ctrl/Cmd+T, or click something that spawns a tab — it pops out into its own window. One click on the toolbar icon turns it off again when you don't want it.

— FEATURES —

• Automatic — new tabs become new windows, no shortcut to remember.
• One-click toggle — enable or disable instantly from the toolbar button; the icon shows the current state.
• Window position — cascade (default), match the parent window's location, or always maximize.
• Window size — keep Chrome's default, or set a custom width × height.
• Focus control — open new windows in front, or quietly in the background.
• Link-only mode — only act on tabs opened from a link or page, and leave blank Ctrl/Cmd+T tabs alone.
• Skip pinned tabs — never tear your pinned tabs out of their window.
• Minimum-tabs threshold — only split a tab out once the window already has at least N tabs, so small windows stay intact.
• URL exclusions — keep chosen sites as tabs using simple substring or wildcard patterns (e.g. *://mail.google.com/*).
• Syncs your settings across devices via your Chrome profile.
• Six languages: English, Spanish, Italian, Japanese, Norwegian (Bokmål and Nynorsk).

— PRIVACY —

No accounts. No tracking. No data collection. The extension only stores your settings, and it requests a single permission ("storage") to remember them. It does not read your browsing history or the content of any page.

— GOOD TO KNOW —

Open the options page (right-click the icon → Options, or chrome://extensions → Details → Extension options) to configure everything above. Changes save instantly.

Built with Manifest V3.
```
